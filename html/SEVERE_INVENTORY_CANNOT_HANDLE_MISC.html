Cannot handle misc. inventory

This is a critical problem and should be reported to turt2live immediatly (email: turt2live@turt2live.com). This error is produced when AntiShare cannot deal with an inventory. A 'misc inventory' is an inventory created by AntiShare but has no code to produce it in a 'custom' way.

How do I fix this?

You must contact turt2live to get this error fixed. It is STRONGLY encouraged you do so.
